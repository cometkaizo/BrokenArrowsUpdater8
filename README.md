## Minecraft Mod Updater
This is a Minecraft mod auto-updater for friends to use while playing on a server. 
Due to the frequent updates of the Origins mod, it became a pain to send the mod to every server member for them to install.
This program eliminates the hassle of manual installation by updating players' mods according to a modlist on a server.
As long as the players run this program before launching, their mods will always be up-to-date.

Eventually, I replaced this system with an even more automated one, making use of java agents that run automatically whenever the game is launched, thus eliminating the need to run an updater program manually.
